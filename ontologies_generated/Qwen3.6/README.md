## 2. Design Philosophy — IDEArq-Inspired Lifecycle Modeling

Qwen3.6 adopts an **IDEArq-inspired** design that centers on archaeological object modeling with explicit lifecycle support, differing from both deepseek-v4-pro (event-centric) and K2.6 (object-centric with deep taxonomy):

| Aspect | Deepseekv4 (Event-centric) | K2.6 (Object-centric) | Qwen3.6 (IDEArq-inspired) |
|---|---|---|---|
| **Core entities** | Events (ProductionEvent, UseEvent, DepositionEvent, RecoveryEvent) | Objects (NaturalObject, HumanMadeObject, Artefact, Structure) | Objects + Lifecycle events (ObjectBiography, ManufacturingEvent, DisplacementEvent) |
| **Hierarchy depth** | Flat (objects + events as siblings) | Deep (3-level taxonomic tree) | Moderate (2 levels: ArchaeologicalObject → NaturalObject/HumanMadeObject → subtypes at 0.7) |
| **Lifecycle modeling** | Implicit through event classes | Implicit through object attributes | Explicit through ObjectBiography class and lifecycle event types |
| **Spatial integration** | Minimal | Moderate (StratigraphicSequence, NegativeFeature) | Extensive (ArchaeologicalPlace, Site, ActivityZone, GeoSPARQL alignment) |
| **Temporal integration** | Minimal | Moderate | Extensive (ArchaeologicalTimeSpan, CulturalPeriod, OWL-Time alignment) |
| **Stratigraphic integration** | DepositionEvent, RecoveryEvent | StratigraphicSequence, Paleosol | StratigraphicContext, HarrisMatrix, FormationEvent |
| **Disjointness axioms** | None | NaturalObject disjointWith HumanMadeObject | Same as K2.6 + Artefact disjointWith Structure at 0.7 |

### Key design decision

The IDEArq-inspired model treats the **archaeological object** as the primary ontological anchor while explicitly modeling its **lifecycle** through interconnected events (manufacturing, use, repurposing, deposition, recovery, conservation). This mirrors the IDEArq conceptual model where objects are understood through their biographical trajectory rather than as static entities or as pure event chains.

### Object classification (inspired by IDEArq UML)

```
ArchaeologicalObject (crm:E19_Physical_Object)
├── NaturalObject
│   ├── AbioticObject (stone, mineral, sediment)
│   └── BioticObject (bone, shell, plant remains)
└── HumanMadeObject
    ├── Artefact (portable functional object)
    ├── Structure (non-portable construction)
    └── ArtisticExpression (aesthetic/symbolic object)
```
