# Competency Questions — Archaeological Object (Event-Centric Design)
> **Generated by:** Kimi-2.6
> **Date:** 2026-05-26
> **Block:** Archaeological Object
> **Total CQs:** 30
> **Source:** 14 articles in `docs/objetoArqueologico/` + ontological gap analysis
> **Design philosophy:** Event-centric — the archaeological event is the primary ontological anchor. Objects participate in events; events carry the semantic weight. Object biography is told as an ordered sequence of events, not as object-attributed properties.

---

## Bloque 1: Eventos de producción y manufactura

### CQ-OBJ-01
**Question:** Which manufacturing events are documented for objects recovered from a given stratigraphic unit, and what technological procedures (chaine operatoire) were employed in each event?
- **Ontology modules required:** Production Module, Archaeological Object Module
- **Possible ontology reuse:** CIDOC CRM (`E12_Production`, `E29_Design_or_Procedure`), CRMarchaeo (`A8_Stratigraphic_Unit`)
- **Source:** Garcia-Rovira (csa_vol_23_2015) — process ontology of manufacturing; Guerra (2003) — XRF signatures of manufacturing techniques

### CQ-OBJ-02
**Question:** What raw material extraction events supplied the materials later used in manufacturing events attributed to objects found at a site?
- **Ontology modules required:** Production Module, Materiality Module, Spatial Module
- **Possible ontology reuse:** CIDOC CRM (`E12_Production`, `E53_Place`, `E57_Material`), GeoSPARQL
- **Source:** Pernicka et al. — lead isotope tracing from mine to object; Guerra (2003) — gold source identification

### CQ-OBJ-03
**Question:** In which manufacturing events did the same chaine operatoire produce both objects classified as human-made and objects whose raw materials remained unmodified by human action beyond selection?
- **Ontology modules required:** Production Module, Taxonomy Module
- **Possible ontology reuse:** CIDOC CRM (`E12_Production`, `E22_Human-Made_Object`, `E19_Physical_Object`)
- **Source:** Garcia-Rovira — nature/culture entanglement; the ecofact/artefact boundary is often ambiguous in production contexts

### CQ-OBJ-04
**Question:** What is the temporal gap between the manufacturing event of an object type and the earliest documented use event of a specific instance of that type?
- **Ontology modules required:** Production Module, Temporal Module, Object Biography Module
- **Possible ontology reuse:** CIDOC CRM (`E12_Production`, `E7_Activity`, `E52_Time-Span`), OWL-Time
- **Source:** BF03376602 — ceramics remain in use 15+ years after their type ceases manufacturing; the manufacturing date range of the type is systematically different from the use date range of the instance

### CQ-OBJ-05
**Question:** Which manufacturing events employed non-local raw materials that had previously moved through circulation events from their geological source to the production locus?
- **Ontology modules required:** Production Module, Circulation Module, Materiality Module
- **Possible ontology reuse:** CIDOC CRM (`E12_Production`, `E9_Move`, `E53_Place`, `E57_Material`)
- **Source:** Gill — objects travel through trade, gift exchange, and looting; Guerra — provenance chain from raw material to finished object

## Bloque 2: Eventos de uso, reúso y transformación funcional

### CQ-OBJ-06
**Question:** Which use events extended an object's active life beyond the manufacturing date range of its typological class, and what functional role was the object serving in those use events?
- **Ontology modules required:** Use Module, Temporal Module, Object Biography Module
- **Possible ontology reuse:** CIDOC CRM (`E7_Activity`, `E55_Type`, `E52_Time-Span`)
- **Source:** BF03376602 — the use range of a specific object is not the same as the manufacturing range of its type

### CQ-OBJ-07
**Question:** What reuse events transformed an object from its original intended function to a substantially different functional role, and what trace evidence of that functional transformation remains on the object?
- **Ontology modules required:** Use Module, Object Biography Module
- **Possible ontology reuse:** CIDOC CRM (`E7_Activity`, `E81_Transformation`, `E3_Condition_State`)
- **Source:** "Things in the Eye of the Beholder" — objects accumulate multiple meanings through cycles of use and reuse; Kopytoff's cultural biography of things

### CQ-OBJ-08
**Question:** Which repair events restored or extended an object's function during its active use-life, and how are those repair events distinguished from post-excavation conservation treatments?
- **Ontology modules required:** Use Module, Conservation Module, Object Biography Module
- **Possible ontology reuse:** CIDOC CRM (`E81_Transformation`, `E7_Activity`, `E3_Condition_State`)
- **Source:** ICCROM — repair is a deliberate intervention during use-life, distinct from post-recovery conservation

### CQ-OBJ-09
**Question:** What object agency events — where the form, materiality, or cultural meaning of an object shaped human behavior independently of the human actor's original intention — are documented in the use-life of an object?
- **Ontology modules required:** Use Module, Object Meaning Module
- **Possible ontology reuse:** CIDOC CRM (`E7_Activity`, `E39_Actor`), CRMinf (`I4_Proposition_Set`)
- **Source:** Gosden — "What do objects want?" Objects shape human behavior through form, genealogy, source, and effect

### CQ-OBJ-10
**Question:** Which gradual transformation events altered an object's physical form or function continuously over time, rather than through discrete modification events?
- **Ontology modules required:** Object Biography Module, Material Transformation Module
- **Possible ontology reuse:** CIDOC CRM (`E81_Transformation`, `E3_Condition_State`), CRMsci (`S4_Observation`)
- **Source:** Garcia-Rovira — objects are always "becoming" rather than static; continuous change is the norm

### CQ-OBJ-12

## Bloque 4: Eventos de deposición y post-deposición

### CQ-OBJ-16
**Question:** What deposition events led to the entry of objects into the archaeological record, and were those depositions intentional (ritual discard, burial) or accidental (loss, abandonment)?
- **Ontology modules required:** Deposition Module, Stratigraphy Module
- **Possible ontology reuse:** CIDOC CRM (`crmarchaeo:A4_Stratigraphic_Genesis`, `E5_Event`), CRMarchaeo (`A8_Stratigraphic_Unit`)
- **Source:** BF03376602 — deposition is the terminal event of the object lifecycle before post-depositional processes begin

### CQ-OBJ-17
**Question:** What post-depositional alteration events (patination, corrosion, weathering, bioturbation) affected an object after its deposition event, and how did those alteration events change the object's condition state?
- **Ontology modules required:** Post-Deposition Module, Material Transformation Module
- **Possible ontology reuse:** CIDOC CRM (`E3_Condition_State`, `E5_Event`), CRMarchaeo (`A5_Stratigraphic_Modification_Event`)
- **Source:** Holtorf — patina as physical surface alteration; ICCROM — weathering and degradation processes

### CQ-OBJ-18
**Question:** Which fragmentation events broke an object into multiple disconnected fragments after its deposition event, and how do those fragmentation events differ from intentional part-whole composition events during manufacture?
- **Ontology modules required:** Post-Deposition Module, Partonomy Module, Production Module
- **Possible ontology reuse:** CIDOC CRM (`E3_Condition_State`, `E19_Physical_Object`, `P46_is_composed_of`)
- **Source:** BF03376602 — archaeological ceramics are predominantly recovered as fragments (sherds); fragmentation is accidental, not intentional composition

### CQ-OBJ-19
**Question:** What recovery events excavated objects from their depositional contexts, and what stratigraphic units and excavation process units were involved in each recovery event?
- **Ontology modules required:** Recovery Module, Stratigraphy Module
- **Possible ontology reuse:** CIDOC CRM (`crmarchaeo:A1_Excavation_Process_Unit`, `crmarchaeo:A8_Stratigraphic_Unit`)
- **Source:** BF03376602 — recovery marks the transition from archaeological deposit to analyzed object

### CQ-OBJ-20
**Question:** Which taphonomic processes in the depositional environment created patina or surface alteration that was later culturally valued as evidence of authenticity or pastness?
- **Ontology modules required:** Post-Deposition Module, Object Meaning Module
- **Possible ontology reuse:** CIDOC CRM (`E3_Condition_State`, `E5_Event`), CRMinf (`I4_Proposition_Set`)
- **Source:** Holtorf — Riegl's age-value: patina is both a physical condition state and a cultural value marker; the museum debate about cleaning patina reflects its dual nature

## Bloque 5: Eventos de análisis científico y datación

### CQ-OBJ-21
**Question:** What analytical encounter events applied non-invasive or micro-destructive techniques (XRF, ICP-MS, SEM-EDS, Raman spectroscopy) to an object, and what material assignments resulted from each analytical encounter?
- **Ontology modules required:** Analytical Module, Materiality Module
- **Possible ontology reuse:** CRMsci (`S19_Encounter`, `S21_Measurement`), CIDOC CRM (`E17_Type_Assignment`, `E57_Material`)
- **Source:** Guerra, "Non-invasive studies", Pernicka et al. — material characterization is a scientific event, not an intrinsic property

### CQ-OBJ-22
**Question:** What chronometric dating events (radiocarbon, U-series, K-Ar, Ar-Ar, TL, OSL) produced chronological assignments for an object or its context, and what are the calibrated age intervals with associated uncertainty ranges?
- **Ontology modules required:** Chronometric Module, Temporal Module
- **Possible ontology reuse:** CRMsci (`S19_Encounter`, `S21_Measurement`), CIDOC CRM (`E17_Type_Assignment`, `E52_Time-Span`), OWL-Time
- **Source:** Schwarcz — radiocarbon and other chronometric methods; "2949238" — overview of dating techniques

### CQ-OBJ-23
**Question:** Which analytical encounter events used lead isotope ratios or trace element analysis to determine the geographic source of an object's raw materials, and how do those provenance determinations link to known geological source regions?
- **Ontology modules required:** Analytical Module, Provenance Module, Spatial Module
- **Possible ontology reuse:** CRMsci (`S19_Encounter`, `S21_Measurement`), CIDOC CRM (`E53_Place`), GeoSPARQL
- **Source:** Pernicka et al. — lead isotope ratios (206Pb, 207Pb, 208Pb vs 204Pb) as gold standard for metal provenance; Guerra — XRF elemental composition

### CQ-OBJ-24
**Question:** What terminus post quem and terminus ante quem events establish the chronological boundaries for an object's deposition, and how do those boundary events relate to manufacturing events and use events in the object's lifecycle?
- **Ontology modules required:** Chronometric Module, Temporal Module, Object Biography Module
- **Possible ontology reuse:** CIDOC CRM (`E52_Time-Span`, `E12_Production`, `E7_Activity`), OWL-Time
- **Source:** BF03376602 — TPQ is "the cornerstone of all archaeological dating"; TAQ provides the upper bound; both must be distinguished from manufacturing and use ranges

### CQ-OBJ-25
**Question:** Which analytical encounter events on an object produced conflicting material assignments or chronological assignments when different techniques were applied, and how were those conflicts resolved?
- **Ontology modules required:** Analytical Module, Inference Module
- **Possible ontology reuse:** CRMsci (`S19_Encounter`), CRMinf (`I4_Proposition_Set`, `I2_Belief`), CIDOC CRM (`E17_Type_Assignment`)
- **Source:** Rowe, Schwarcz — multiple dating techniques may produce conflicting chronologies; different techniques yield different assignments

## Bloque 6: Eventos de conservación, interpretación y significado

### CQ-OBJ-26
**Question:** What conservation treatment events were applied to an object after its recovery event, and what conservation decision events — involving stakeholder values, risk assessment, and ethical negotiation — preceded each treatment?
- **Ontology modules required:** Conservation Module, Post-Excavation Module
- **Possible ontology reuse:** CIDOC CRM (`E7_Activity`, `E5_Event`), CRMsci (`S4_Observation`)
- **Source:** ICCROM — conservation is not purely technical; it involves negotiating competing values (authenticity, access, research, preservation)

### CQ-OBJ-27
**Question:** Which typological assignment events classified an object under a specific typological scheme, and what diagnostic criteria (morphological, technological, stylistic, functional) were invoked in each assignment event?
- **Ontology modules required:** Classification Module, Object Biography Module
- **Possible ontology reuse:** CIDOC CRM (`E17_Type_Assignment`, `E55_Type`, `E54_Dimension`), SKOS, Getty AAT
- **Source:** Rowe — typology is the foundation of archaeological classification; classification is always an assignment, not an intrinsic property

### CQ-OBJ-28
**Question:** What object biography events — narrative constructs told by specific interpreters — have been constructed for an object, and how do those biography events differ from the measurable lifecycle events of the same object?
- **Ontology modules required:** Object Biography Module, Object Meaning Module, Inference Module
- **Possible ontology reuse:** CIDOC CRM (`E5_Event`), CRMinf (`I4_Proposition_Set`), PROV-O
- **Source:** "Things in the Eye of the Beholder" — object biography is a narrative construct, not an objective physical history; multiple competing biographies may exist for the same object

### CQ-OBJ-29
**Question:** Which pastness events — culturally constructed experiences of an object as being from the past — have been documented for an object, and how do those pastness events relate to the object's physical condition states (patina, weathering) rather than to its measured chronological age?
- **Ontology modules required:** Object Meaning Module, Post-Deposition Module
- **Possible ontology reuse:** CRMinf (`I4_Proposition_Set`), CIDOC CRM (`E3_Condition_State`, `E52_Time-Span`)
- **Source:** Holtorf — pastness is an aesthetic, emotional, cultural experience of antiquity, NOT reducible to measured age; "What they have taken away is the age of the paint"

### CQ-OBJ-30
**Question:** What cultural significance assignment events attributed symbolic, ritual, or aesthetic meaning to an object, and how do those significance events relate to functional use events, reuse events, and ritualization events in the object's biography?
- **Ontology modules required:** Object Meaning Module, Use Module, Object Biography Module
- **Possible ontology reuse:** CIDOC CRM (`E17_Type_Assignment`, `E7_Activity`, `E55_Type`), CRMinf (`I4_Proposition_Set`)
- **Source:** Pollard — tabua (Fijian whale teeth) as objects whose value derives from meaning, not material; Holtorf — ritualization transforms profane objects to sacred status
